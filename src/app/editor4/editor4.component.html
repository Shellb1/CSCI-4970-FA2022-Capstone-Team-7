Feature Comparison

Library Features
PDF.js
  • Free to use.
  • Open Source.
  • Not as many features as the others. Will have to build key features using library.
  • Library is in JavaScript, so able to be embeded using any modern web framework.
  • License on GitHub explains details of use.
PSPDFKit
  • More features than PDF.js.
  • Easy to embed in almost every modern web framework, including React.js.
  • Not open source.
  • Have to pay a licensing fee. Commercial support.
PDFTron
  • Has the most features. Fits most of what Buildertrend is looking for.
  • Easy to embed.
  • Like PSPDFKit, have to pay a licensing fee. Most fully featured application might mean it's the most expensive.
  • Not open source.
