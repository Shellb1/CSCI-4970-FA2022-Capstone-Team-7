This is a project implementing client-side PDF annotations for Buildertrend Inc. This project is a part of the CSCI 4970
Capstone Project for Computer Science students at the University of Nebraska-Omaha.
This project was built by Andrew Shellberg, Colin Brammeier, Brody Himmelberg, and Chris Kiihne.
Features
Opening and Editing a PDF using PDF.js
Opening and Editing a PDF using PSPDF Kit
Opening and Editing a PDF using PDFTron
